Import DIDs

{{/* The form is hidden once an import starts, replaced by the progress panel. On done, the user can click "Import more" to reset. */}}

Generates one DID per integer in [start, end]. End defaults to start. Max 10,000 per import.

Numbers can be in any common format (with or without +, spaces, dashes, parentheses). Anything that doesn't parse as a 6–15-digit number is skipped with a warning.

Required column: e164. Optional per-row override columns: country_iso, did_type, supplier_id, channel_cap. Headerless files work too — columns are read in that order. Per-row values override the form defaults.
