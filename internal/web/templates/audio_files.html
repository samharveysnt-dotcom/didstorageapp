Upload audio

Accepted: mp3, wav, m4a, ogg, opus, flac, aac, webm, slin, ulaw, alaw, gsm. Max {{div .MaxUploadBytes 1048576}} MB. Stereo is downmixed; high sample rates are resampled to 8 kHz mono 16-bit.

Bulk upload audio

Pick several files at once. Each gets converted to 8 kHz mono and saved as <prefix>-1, <prefix>-2, … If an existing prefix has e.g. 1–3, the batch continues from -4. Optionally drop the whole batch straight into an existing audio group.

Files added to a group can serve as a rotating set on an audio_group-kind reserved DID (random-no-repeat per call). Manage groups.

Rename audio

Renaming only changes the label shown in dropdowns. The on-disk filename stays the same so any DID already reserved with this clip keeps working without a reload.
